# 우리가족 거우리

### 개요
- - - 
**"우리가족 거우리"**
화장실에서 사용하는 거울로, 어린이들에게는 양치, 손 씻기에 대한 좋은 습관
기르기를 도와주면서 거울과의 상호작용을 통해 화장실에 대한 거부감이 들지 않게 해줍니다.
성인에게는 개인별 맞춤 화면을 통해 편의 제공과 함께
아이의 양치와 손 씻기 습관의 과정을 사진과 메세지를 통해 공유합니다.
